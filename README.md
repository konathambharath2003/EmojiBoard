# EmojiBoard
An integration of Emoji recommendation embedded model on open source floris board keyboard project.
### 📲 How to Install
1. Go to the **Releases** section on the right side of this page.
2. Download the `keyboardv1.apk` file.
3. Open it on your Android phone. (You need to allow "Install from unknown sources" in your settings).
4. Follow the settings of the App to enable it has default keyboard.
